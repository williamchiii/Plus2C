# +2°C

An interactive Swift Playgrounds app exploring the consequences of a +2°C rise in global temperature.

## Features

- **Sea Level Rise Explorer** — Visualize sea level changes across temperature scenarios with an interactive flood simulator
- **Risk Snapshot** — Current and future climate risk outlooks
- **Action Checklist** — A personal climate action plan
- **Time Machine** — View the Earth across past and future timelines
- **Interactive Globe** — 3D globe with real-time warming visualization

## Requirements

- Swift Playgrounds 4+ or Xcode 15+
- iPadOS 16+ / macOS 13+

## Getting Started

1. Clone the repository
2. Open `Plus2C.swiftpm` in Swift Playgrounds or Xcode
3. Run the app
